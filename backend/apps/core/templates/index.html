welcome to μblog!

μblog is a reference application built with Django and Vue.js. It is a reference project that implements a simple microblogging application.

Write your first μblog post Go to μblog posts Read more about μblog

μblog is built with open-source technologies

Django is the web framework that powers μblog. It handles the data persistence and server-rendered templates.

μblog is built with Vue 3 and takes advantage of the composition API and TypeScript integration. Vue is also used for interactivity in Django server-rendered templates

Django REST Framework powers the REST API in the Django application. This allows the frontend application to query data from μblog.

Quasar is a Vue Framework that μblog uses. It offers a component library that follows the Material UI spec and allows for multiple built targets including SPA, PWA and SSR.
